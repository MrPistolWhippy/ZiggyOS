# 👑 ZIGGY OVERLORD OS v1.0

A high-performance, bare-metal operating system kernel compiled for the ARM Cortex-M4 architecture (NXP LPC4330). Bypasses desktop library wrappers to establish absolute control over raw silicon memory grids.

## 🛰️ System Architecture Modules
* **Hardware Timing Matrix**: Automated `SysTick` vector interrupt loops running natively on a solid clock cycle.
* **2D Radar Sweep Canvas**: Microscopic geometric concentric line vectors with active red target blip positioning tracking.
* **16-Byte Hex Scanner Matrix**: Direct memory pointer extraction rendering live sequential register blocks (`MX0` and `MX1`) scrolling dynamically via physical rotary dial pins.

## 🚀 Local Build Toolchain Pipelines
* Rebuild and link source layers: `make clean && make`
* Run pre-flight debugger validation checks: `sh verify_first.sh`
* Containerize firmware for deployment: `python3 pack_firmware.py`
